# SmartBudget

### The Smart Personal Budget Management Application

It is an intelligent mobile solution designed to simplify financial tracking and improve spending habits. Here is a summary of its core features and functionality:

AI-Powered Insights: Automatically categorizes expenses and analyzes user behavior to provide personalized recommendations for saving money.

Flexible Data Entry: Users can input transactions manually or upload CSV files for bulk automated processing.

Visual Monitoring: Includes an interactive dashboard with dynamic charts to visualize trends in income, spending, and savings.

Proactive Management: Features real-time alerts for budget limits and a goal-tracking system with progress indicators to keep users on path.

Accessibility: Offers a bilingual interface in both Arabic and English to accommodate a diverse user base.


## Firebase Setup:
For security reasons, the google-services.json file is ignored. To run the project locally:

Create your own project on Firebase Console.

Download your google-services.json file.

Place it inside the android/app/ directory.
